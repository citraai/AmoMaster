# HerSpecialist - 要件定義書

## 1. プロダクト概要

### 1.1 名称
**HerSpecialist**

### 1.2 コンセプト
ユーザーを「最高のパートナー」へ育成するスパルタ・コンシェルジュアプリ。
毒舌だが愛情深い「恋愛マスター」AIが、ユーザーを叱咤激励しながらパートナーを幸せにするためのサポートを行う。

### 1.3 ターゲットユーザー
- パートナー（彼女/彼氏）をもっと幸せにしたいと思っている人
- 自分の恋愛スキルを向上させたい人
- パートナーの好みや発言を忘れがちな人

---

## 2. コア機能

### 2.1 嗜好品DB / 言霊ログ
**目的**: パートナーの「好き」や「発言」を爆速で記録

**仕様**:
- 3タップ以内で記録完了
- カテゴリ: 食べ物、場所、アクティビティ、プレゼント、言葉など
- タグ付け機能
- 日時自動記録
- 検索・フィルタ機能

### 2.2 恋愛マスターAI
**目的**: 記録に基づき、毒舌かつ厳格にユーザーを指導

**キャラクター設定**:
- 口調: 厳しめ・毒舌・愛情深い
- スタンス: 常にパートナー至上主義
- 役割: ユーザーの「甘え」「怠慢」を許さない指導者

**機能**:
- 毎日の「喝」メッセージ
- 記念日・イベントのリマインド＆プラン提案
- パートナーの好みに基づくプレゼント提案
- 行動へのフィードバック

### 2.3 地雷回避シミュレーター
**目的**: 提案プランを過去の「NGログ」と照合し警告

**仕様**:
- NGログ: パートナーが嫌がったこと、失敗したデートなどを記録
- プラン提案時に自動照合
- リスクスコア表示
- 代替案の提示
